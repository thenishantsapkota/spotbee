# spotbee

This is a module that spits out YouTube URLs from Spotify Playlist URLs

**Why use this?**

It is asynchronous which makes it compatible to use with Discord Bots.

**Installation**

```bash
pip install spotbee
```

**Examples*

Check the `examples` folder.
